# Soft Toon Renderer

OpenCV를 이용하여 입력 이미지를 부드러운 만화 스타일로 변환하는 프로그램입니다.  
이미지의 색상을 단순화하고 외곽선을 추출한 뒤, 채도와 색온도를 조절하여 따뜻하고 부드러운 분위기를 표현하도록 구현했습니다.

## 실행 방법

아래 명령어를 터미널에서 실행하면 결과 이미지를 생성할 수 있습니다.

```bash
python cartoonize.py --input input/good.jpg --output output/good_result.png
python cartoonize.py --input input/bad.jpg --output output/bad_result.png
```bash

## 알고리즘 설명

본 프로그램은 다음과 같은 순서로 이미지를 처리합니다.

1) bilateral filter를 반복 적용하여 이미지 내부를 부드럽게 만듭니다.
2) k-means 기반 color quantization을 사용하여 색의 수를 줄이고 평면적인 색감을 만듭니다.
3) HSV 색공간에서 채도와 밝기를 조정하여 색을 조금 더 생동감 있게 만듭니다.
4) BGR 채널별 가중치 조정을 통해 전체적으로 따뜻한 색감을 더합니다.
5) grayscale 변환, median blur, adaptive threshold를 통해 외곽선을 추출합니다.
6) 최종적으로 외곽선 마스크를 색상 보정된 이미지에 적용하여 만화 스타일 결과 이미지를 생성합니다.

## 잘 표현되는 이미지 예시

- 아래 이미지는 하늘, 건물, 도로, 계단과 같이 비교적 큰 색 영역이 포함되어 있어 색상 단순화 효과가 잘 드러납니다.
- 또한 장면의 주요 구조가 직선과 큰 면 위주로 이루어져 있어, 외곽선이 비교적 안정적으로 유지됩니다.
- 그 결과 전체 장면이 보다 평면적이고 부드러운 만화풍으로 변환되었으며, 색온도 보정으로 따뜻한 분위기도 잘 확인할 수 있었습니다.

## 잘 표현되지 않는 이미지 예시

- 아래 이미지는 나뭇잎, 가지, 그림자, 빛의 방향 변화 등 세부 텍스처가 매우 많아 만화풍으로 단순화하기 어려운 장면입니다.
- 색상 단순화 이후에도 배경의 복잡함이 충분히 줄어들지 않았고, 외곽선 역시 세밀한 영역에서 많이 생성되어 결과가 다소 복잡하게 보입니다.
- 따라서 부드럽고 정돈된 만화 스타일 효과를 확인하기에는 상대적으로 불리한 이미지였습니다.

## 한계점

본 알고리즘은 OpenCV 기반의 전통적인 영상처리 기법을 사용하므로 다음과 같은 한계가 있습니다.

- 이미지의 의미를 이해하지 못하고 픽셀 단위 정보만 사용합니다.
- 복잡한 배경이나 세부 텍스처가 많은 이미지에서는 불필요한 외곽선이 많이 생성될 수 있습니다.
- 조명과 그림자의 영향으로 실제 물체 경계가 아닌 부분도 외곽선으로 검출될 수 있습니다.
- 색상 단순화와 톤 보정을 통해 만화 같은 분위기는 만들 수 있지만, 특정 애니메이션의 화풍을 정확히 재현하기는 어렵습니다.
- 이미지마다 적절한 파라미터 조정이 필요하며, 모든 이미지에서 동일하게 좋은 결과를 보장하지는 않습니다.

## 결론

- 비교적 큰 색 영역과 명확한 구조를 가진 이미지에서는 색상 단순화와 외곽선 추출 효과가 잘 나타났습니다.
- 반면, 숲과 같이 세부 텍스처와 복잡한 조명이 많은 장면에서는 불필요한 윤곽선이 증가하여 결과가 덜 정돈된 모습으로 나타났습니다.
- 이를 통해 전통적인 영상처리 기반 만화 렌더링은 이미지의 구조와 복잡도에 따라 결과 차이가 크게 나타난다는 점을 확인할 수 있었습니다.
